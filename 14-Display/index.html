DISPLAY

No cambia el tipo de elemento, cambia cómo se visualizan los elementos en un sitio web

Los tipos de elementos html:

  • Bloque: block, elemento que empieza en una nueva línea, ocupa todo el ancho disponible de la pantalla, a menos que le indiquemos tamaño con width
    1. Encabezados h1-h6
    2. Párrafos
    3. Formularios
    4. Listas ul, ol, dl, y sus li
    5. div
    6. table, td, tr, th
  • En línea: inline, ocupan solamente el tamaño de su content, no les podemos dar ancho ni alto y van uno a continuación del otro
    1. a enlaces
    2. Imágenes
    3. span

    VALORES DEL DISPLAY

    • none: no se muestra. No deja el hueco del elemento que se oculta
    • visibility: hace lo mismo que none pero deja el hueco del elemento
    • block: se muestre como si fuera un elemento en bloque
    • inline: se muestre el elemento como si fuera un elemento inline
    • inline-block: se muestre como si fuera medio bloque medio inline. Los elementos se sitúan uno a continuación del otro, ocupando lo que tienen que tienen ocupar, pero se le puede dar tamaño y ancho
