POSITION

Permite posicionar elementos html en cualquier lugar de la pantalla

Valores:

  • static: es el valor por defecto. siguen el flujo normal de la página según su colocación
  • relative: se puede mover, se mueve desde su posición en el flujo normal. top, bottom, left, right
    • El hueco del elemento movido se mantiene!!!
  • absolute: se puede mover, hay que comprobar si tiene padre y si este padre está posicionado (significa que tiene un valor de position cq distinto a static) se mueve dentro del padre, y si no tiene padre o tiene y no está posicionado (el padre), se mueve en relación a la pantalla (body) Se mueve con las propiedades: top, bottom, left, right. El elemento movido no deja hueco.
  • fixed: se puede mover, con top, bottom, left, right, se mueve en relación a la pantalla, y se queda fijo. El hueco del elemento movido no se mantiene.
  • sticky: se comporta como relative y como fixed. Se fija con top, bottom, left, right.
